## Performance Tips: Getting the Most from Gemini 3 Pro

### 1. **Be Specific, But Not Over-Prescriptive**

**Good prompt:**
*"Create a user profile page with avatar upload, bio editing, and social links. Use a modern card layout with smooth transitions."*

**Over-prescriptive prompt:**
*"Create a div with className 'profile-container' and inside it create another div with className 'avatar-wrapper'..."*

Gemini 3 Pro excels when you describe **what** you want, not **how** to build it.

### 2. **Use Deep Think for Hard Problems**

For complex debugging or architectural decisions, explicitly ask:
*"Use Deep Think mode to analyze this performance issue"*

### 3. **Leverage Multimodal Input**

Instead of describing a UI in words, upload a screenshot or sketch. Gemini 3 Pro will understand it better than a text description.

### 4. **Let It See Your Whole Codebase**

With 1M token context, you can ask:
*"Review my entire project and suggest performance optimizations"*

It'll analyze everything and provide comprehensive insights.

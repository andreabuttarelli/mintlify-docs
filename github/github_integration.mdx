## ‚ö°Ô∏èüêô Connect and Use GitHub with Teta

<Steps>
  <Step title='Install the "Teta.so" GitHub App'>
    To allow Teta to access repositories, you need to install the "Teta.so" GitHub App.
  </Step>
  <Step title="Create a new project from a GitHub repository">
    From the homepage, select the repository from the dropdown, the desired branch, and optionally set the project's base path (default '/' - root).
  </Step>
  <Step title="Connect an existing Teta project to a GitHub repository">
    Go to `Settings > GitHub` and follow the same configuration steps described in point 2.
  </Step>
  <Step title="Project Synchronization">
    Once the project (new or existing) is connected to a GitHub repository, it will automatically synchronize every time there is a push or pull request in the selected branch.
  </Step>
  <Step title="Commit and Pull Request">
    From the top menu, by clicking the 'Push to GitHub' button, a dialog will open allowing you to choose whether to perform a push with a commit or create a pull request. A pull request is recommended to review changes before accepting them.
  </Step>
</Steps>

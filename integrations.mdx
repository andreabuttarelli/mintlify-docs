## What are Integrations?

Integrations in Teta are powered by **MCP (Model Context Protocol) servers**. Think of them as plugins that connect the Teta AI to your existing tools and workflows.

## How to Add an Integration

Most integrations are added by configuring a corresponding MCP server in your project settings.

1.  **Find the integration** you want to add from the list above.
2.  **Follow the setup instructions** in the integration's documentation.
3.  Typically, this involves:
    *   Generating an API key or token from the third-party service.
    *   Adding a new MCP server configuration in **Settings > MCPs**.
    *   Storing any sensitive tokens as Teta secrets.

## Custom Integrations

If you don't see an integration you need, you can build your own. MCP is an open standard, and you can create a custom server for any internal or third-party tool.

- **Learn more about custom MCP servers in the [MCP Servers documentation](./mcp-servers.md).**

## Security

When connecting third-party tools, it's important to follow security best practices:
- **Use dedicated tokens** with the minimum required permissions.
- **Never expose secrets** in your code; use Teta's secret management.
- **Review permissions** regularly and revoke access when no longer needed.

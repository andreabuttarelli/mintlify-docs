## What is Cloud VS Code?

Teta provides a **complete VS Code environment** running in the cloud, accessible directly from your browser. This isn't a simplified editor‚Äîit's the full Visual Studio Code experience with support for extensions, terminal access, and all the tools you're used to working with.

## Why Cloud VS Code?

While AI can handle many coding tasks, sometimes you need **precise control** over your code. Cloud VS Code gives you the best of both worlds:

- **AI-Powered Development**: Let the AI handle repetitive tasks and boilerplate code
- **Manual Control**: Jump into the IDE when you need to fine-tune implementation details
- **Seamless Integration**: Switch between AI assistance and manual editing without friction
- **Collaboration**: Work together with your team in real-time on the same codebase

## How to Access Cloud VS Code

### 1. Open from Project Dashboard

1. Navigate to your project in Teta
2. Click on the **"Open VS Code"** button in the navbar
3. The cloud IDE opens in a new browser tab

### 2. Quick Access

- **Keyboard Shortcut**: Press `Cmd/Ctrl + E` from anywhere in the project
- **Context Menu**: Right-click on any file and select "Open in VS Code"

## Frequently Asked Questions

**Can I use all my favorite VS Code extensions?**

Most extensions work perfectly. Extensions that require deep local file system access or specific OS features may have limitations.

**How does collaboration work?**

Multiple users can access the same workspace simultaneously. You'll see live cursors and changes update in real-time.

**Are my files saved automatically?**

Yes. Files auto-save, and changes are committed to your Git branch. You can also manually save with `Cmd/Ctrl + S`.

**Can I customize themes and settings?**

Yes! Install theme extensions and modify settings just like desktop VS Code. Settings are saved to your workspace.

**What about terminal commands?**

The integrated terminal supports most standard commands. You have full access to installed tools (npm, flutter, git, etc.).

**Is it as fast as local VS Code?**

Performance depends on your internet connection. For most tasks, it's comparable to local development. The cloud environment has the advantage of powerful cloud compute resources.

**Can I work offline?**

No. The cloud VS Code requires an internet connection. However, your code is synced to GitHub, so you can always clone and work locally if needed.

**How is this different from GitHub Codespaces?**

While both provide cloud VS Code, Teta integrates AI assistance, live preview, and framework-specific optimizations (Flutter, React, Svelte) all in one platform.

## Limitations

- **Internet Required**: Cloud environment requires stable internet connection
- **Some Extensions**: Extensions requiring local OS features may not work
- **Performance**: Subject to network latency (typically minimal)
- **Storage Limit**: Each workspace has a **16GB storage limit**. Exceeding this limit will cause the virtual machine to break, making it impossible to edit code (both via AI and manually through VS Code). If this occurs, your workspace will be flagged and inspected to prevent abuse. If the overage was unintentional, please contact support@teta.so

## Security & Privacy

- **No Code Storage**: Your code lives in your GitHub repository, not on Teta servers
- **Temporary Workspaces**: Cloud environments are temporary and isolated
- **Secure Sandboxes**: Each workspace runs in a secure, isolated container
- **GitHub-Backed**: All changes are version controlled through GitHub
